# 1038 Pit Display

The videos are not included in this repository. 

### How to get the videos

1. Go to https://drive.google.com/drive/folders/1dtyTq6Kaw06exxJC8n3lLdv0P9612SkO?usp=sharing and download the folder
2. Create a folder in `/src` named `videos`
3. Place the videos in the `videos` folder you just made **(make sure the individual videos are in the `/src/videos` directory, any other directory or subdirectory will not work)**
